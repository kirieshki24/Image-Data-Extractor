# Image-Data-Extractor Python
 Extracts metadata from images

 ## Запуск Python файла:
 1. Скачать и распаковать zip-архив с исходным кодом
 2. Установить Python 3.12
 3. В командной строке(Windows) ввести следующие команды:
    ```
    chdir ...\Image-Data-Extractor-main (Вместо троеточия вставьте путь до директории с .py файлом)
    python -m venv .venv
    .venv\Scripts\activate
    pip install -r requirements.txt
    python main.py
    ```
